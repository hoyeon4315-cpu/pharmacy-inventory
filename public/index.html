API 키 갱신 방법
  1. data.go.kr 로그인 → 마이페이지 → 개발계정
  2. 인증키 목록에서 현재 키 확인 → 만료 전 [연장신청] 클릭
  3. 새 키가 발급되면 아래 명령어로 Cloudflare에 업데이트:
# Git Bash 또는 터미널에서 실행
export PATH="/c/Program Files/nodejs:/c/Users/duih/AppData/Roaming/npm:$PATH"
wrangler pages secret put DATA_GO_KR_API_KEY
# 프롬프트에 새 키 붙여넣기 후 Enter
